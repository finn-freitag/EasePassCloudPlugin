# Ease Pass Cloud Plugin

## Introduction
This is a plugin for the [Ease Pass](https://github.com/FrozenAssassine/EasePass) password manager. To use this plugin, you need he latest version of [Ease Pass](https://github.com/FrozenAssassine/EasePass) and a running instance of [Ease Pass Cloud](https://github.com/finn-freitag/easepasscloud).

## Installation
Download the latest release or build the plugin yourself. For building it yourself, verify that the reference to EasePassExtensibility.dll is set correct. In Ease Pass, navigate to Settings>Plugins and click on "Add from file" select the EasePassCloudPlugin.dll. In the settings scroll to the bottom and expand "Remote Database Settings" and paste your json configuration from your Ease Pass Cloud instance. Restart Ease Pass and select your cloud database.
